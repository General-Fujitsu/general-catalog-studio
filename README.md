# GENERAL Katalog-Studio 2026/27

Webshop / Konfigurator auf Basis der GENERAL Klimaanlagen-Preisliste 2026/27 (gültig ab 01.07.2026).
Eine einzige Datei (`index.html`), keine Abhängigkeiten, läuft komplett offline im Browser.
Alle Preise netto.

## Auf GitHub veröffentlichen (GitHub Pages)

1. Auf github.com → **New repository** → Name z. B. `katalog-studio`, **Private** oder Public, anlegen.
2. **Add file → Upload files** → `index.html` und diese `README.md` hochladen → **Commit changes**.
3. **Settings → Pages** → unter *Build and deployment* Source: **Deploy from a branch**, Branch: **main** / Ordner **/(root)** → **Save**.
4. Nach 1–2 Minuten ist der Shop erreichbar unter:
   `https://DEIN-BENUTZERNAME.github.io/katalog-studio/`
5. Link auf dem iPhone in Safari öffnen → Teilen → **Zum Home-Bildschirm** – dann startet er wie eine App.

Hinweis: Bei einem **privaten** Repository ist GitHub Pages nur mit GitHub Pro/Team öffentlich schaltbar.
Bei einem öffentlichen Repository kann jeder mit dem Link die Preisliste sehen.

## Eigene Daten

- **Signatur** (Name, Telefon, E-Mail): im Tab *Ausgabe* → Modus *Angebot* → „Signatur bearbeiten".
- **Regionalcenter**: dort ebenfalls über „Regionalcenter verwalten" ergänzbar.
- **Interner Empfänger**: im Modus *Intern* frei eintragbar (Name + optionale E-Mail für den Mail-Button).

Alle Eingaben werden lokal im Browser gespeichert (localStorage), nichts verlässt das Gerät.
